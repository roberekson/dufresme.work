Dufresme.work

Dufresme.work is a clean, lightweight framework based on the concepts of Bootstrap, keeping only the most-needed components with a slimmer style.

This framework is still under development and is not recommended for production use at this time.

Installation

Download the newest development version or clone the repo to your development environment.

Include the CSS in your HTML document’s HEAD: <link rel="stylesheet" href="dufresme.work/css/dufresme.min.css">

Include the JS before the closing BODY tag: <script src="dufresme.work/js/dufresme.min.js"></script>
